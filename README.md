# AI Travel Planner

一个基于AI的旅行规划应用，帮助用户智能规划旅行行程。

## 项目结构

- `frontend/`: Vue 3 + TypeScript 前端应用
- `backend/`: Spring Boot 后端服务
- `docs/`: 项目文档
- `docker/`: Docker配置文件
- `.github/workflows/`: GitHub Actions 工作流配置
